# 리듬베르 — 나중에 추가할 기능 백로그

> 냥이 "나중에 할게 다 기억해줘"라고 한 후보 기능 목록. 우선순위는 자유롭게 조정.

## 골랐던 핵심 후보
- **업적 · 뱃지 컬렉션** — AP / 풀콤보 달성, 연속 플레이, 누적 점수 등 조건 달성 시 뱃지 획득. 레벨/XP 시스템과 연계되는 수집 요소.
- **결과 스코어카드 저장 · 공유** — 결과 화면을 예쁜 카드 이미지(PNG)로 저장/공유. 점수·랭크·정확도·곡 정보 포함.
- **노트 · 테마 커스터마이즈** — 노트 색/스킨, 판정선 모양, 전체 컬러 테마(보라/핑크/블루 등)를 설정에서 변경.
- **구간 연습 모드** — 어려운 구간을 지정해 반복 연습. 속도 조절·해당 구간만 루프 재생.

## 그 외 아이디어
- **곡 미리듣기** — 곡 카드에서 짧게 멜로디 재생.
- **판정 타이밍 그래프** — 결과 화면에 빠름/느림 분포(히스토그램) 표시.
- **데일리 미션 · 출석 보상** — 매일 목표, 달성 시 XP 보너스.
- **모바일 햅틱 진동** — 터치 플레이 시 진동 피드백.
- **mp3 자동 비트 감지 채보** — 업로드한 곡을 실제 비트에 맞춰 채보(현재는 BPM 기반 생성).

## 참고
- 모두 단일 HTML 파일 · 오프라인 환경에서 구현 가능한 범위로 선정.
- 온라인 랭킹/리더보드는 별도 서버가 필요해 현재 구조에선 제외.
